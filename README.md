
# 🐢 Turtle Race

A Python Turtle Graphics racing game where the user places a bet on a turtle color and watches multiple turtles race across the screen with random movement.

## ✨ Features

* 🎨 Multiple colored turtles
* 🎯 User betting system
* 🎲 Random turtle movement
* 🏁 Finish line race logic
* ⚡ Smooth Turtle Graphics animation
* 🕹️ Interactive gameplay

## 🛠️ Built With

* Python 3
* Turtle Graphics
* Random Module
* Loops and Conditionals
